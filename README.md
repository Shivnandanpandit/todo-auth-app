# Todo App with Registration & Login

A simple **Todo application** built using **HTML, CSS, and JavaScript**.  
It has **user registration**, **login**, and **per-user todo lists**, all handled using the browser's `localStorage` (no backend server).

## Features

- User Registration (email + password)
- User Login & Logout
- Separate todo list for each user
- Add new todo items
- Mark todos as complete / incomplete
- Delete todos
- Data stored in `localStorage`

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage for data persistence
- Git & GitHub for version control
- GitHub Pages for hosting (optional)
